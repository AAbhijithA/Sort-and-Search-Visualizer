Quick sort works with the divide and conquer strategy. It divides the array into partitions and put all the values lesser than the partition to left and the rest to the left of the array and then it does the same to the left and right partitions repeatedly. This has an average time complexity of O(N*log(N)) with a constant space unless if we consider stack call space it's O(N). If we choose the wrong partition it leads to being O(N^2). Quick Sort is used to build hybrid Sorting algorithms such as Insort which is highly used in the industries.
