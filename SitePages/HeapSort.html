Heap sort works with the building of Max-Heap or Binary Heap strategy. It divides the array into a tree like structure where the parent is larger than it's children and continuously build it if it's not. It's a comparison based sort and is one of the best sorting algorithms along with Merge Sort and Quick Sort. It has a time complexity of O(N*log(N)) with a constant space unless if we consider stack call space it's O(N). Heap Sort is also used to build Insort along with Quick Sort for the most optimized sorting algorithm.
