Bubble sort works with comparing each value to it's next element to swap with if current is greater along the way. This algorithm then runs the comparison along the whole array N times making it O(N^2) and has constant memory usage. This algorithm is a brute force approach compared to other sorting algorithm intuition wise. It is rarely used in real world applications now and works well in only small datasets.
