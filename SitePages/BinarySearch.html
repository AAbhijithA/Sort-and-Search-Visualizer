Binary search works with searching on a sorted array making it O(log2(N)) by keeping left and right pointers and checking if middle is equal to key or not. If it's more then search to the right of the array else search in thee left of the array by shifting the pointers. This algorithm is useful and an approach for types of datasets that are presorted.
