Insertion sort works with selecting an element in an index and checking in the reverse direction if it is placed accurately whithin the range and it swaps with the elements along the way. This algorithm works with a time complexity of O(N^2) and no extra space taken. It is extremely useful in sorting stream based data where data comes one by one and places the elements in place automatically and outperforms the other algorithms in that use-case and building Hybrid Sorting Algorithms like Insort.
